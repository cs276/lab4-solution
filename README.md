# Books

## Requirements

1. Run `npm i` to install all dependencies.
1. Install `sqlite3`.
1. Create an empty `books.db` file.
1. Run `sqlite3 books.db < setup`

## Development

Run:

```
npm run dev
```

then visit http://localhost:3000/ in your browser.
